# Authentication

Learn how to authenticate your requests.

## API Keys

Our API uses API keys for authentication. Here's how to get started:

1. Generate your API key from the dashboard
2. Include it in your requests
3. Keep it secure

## Authentication Methods

We support several authentication methods:

- API Key Authentication
- OAuth 2.0
- JWT Tokens

## Best Practices

- Never share your API keys
- Rotate keys regularly
- Use environment variables
